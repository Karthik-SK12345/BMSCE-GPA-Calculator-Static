HOW TO MAKE THE MOST OF THIS SITE?

  • BMSCE SGPA / CGPA / PERCENTAGE CALCULATOR


    Hey guys , Here i will be giving a quick overview of the website talking about the features and how to use it the right way.This is a SGPA , CGPA and Percentage calculator. Here the students of all branches of BMSCE can calculate SGPA and CGPA simutaneusly and save, update and delete the data. He can login to view his stored data anytime.

  • INDEX PAGE (SIGN UP / LOGIN)


    This is the Page where a user will land when he opens the site and he /she needs to enter proper NAME nad USN and enter his chosen email and Password to sign up.

  • INDEX PAGE (SIGN UP / LOGIN) - CREDENTIALS


    As shown above user needs to enter his credentials and sign up. Next time while login he needs to remember his Password. You can also retrieve your account anytime if you forgot your password.

  • HOMEPAGE LAYOUT


    After login / signup every user will be on this homepage shown above from where they can navigate to their respective menu to perform their Calculations.

  • SGPA CALCULATOR DROPDOWN


    In case the user wishes to calculate his SGPA of a particular semester they can select it from the navbar dropdown or from the homepage cards.

  • SGPA CALCULATOR LAYOUT


    This is how any SGPA calculator layout looks after you select the respective semester you can choose from the branches given.

  • SGPA CALCULATOR RESULTS


    Here the user is asked to enter his Grade secured from dropdown given below each subject and also the year of passing and then hit calculate to get your sgpa and then user also has an option to save the data below and download.

    NOTE : Here suppose any student has secured "F" Grade he can select "F" Grade from select option and save data for now. Later if the student clears those Subjects through either Fast-Track or Make-up he can just update his grade and year of passing and along with sgpa.

  • SGPA CALCULATOR STORED DATA


    If the user has saved / updated his data it shows up right below his name so its easily available for him to update later.

  • CGPA CALCULATOR DROPDOWN


    If the user wishes to calculate his CGPA he can select it from the navbar and navigate to his respective branch / department.

  • CGPA CALCULATOR LAYOUT


    This is how any CGPA calculator layout looks after you select the respective department. Here the user is asked to enter his respective semester sgpa. And calculate so here the user can update his data as when the semester results are released and download the same as well.

  • CGPA CALCULATOR RESULTS


    So once the SGPA is entered they can see the Updated CGPA and also corresponding Percentage.

  • CGPA CALCULATOR SAVED DATA


    If the user has saved / updated his data it shows up right below his name so its easily available for him to update later.

  • PERCENTAGE CALCULATOR


    There is also a separate option for Percentage Calculation where user can simply enter his cgpa and convert to check their Percentage.

  • MY PROFILE DROPDOWN


    This is the user profile dropdown. Here the user can check for all his saved and updated data of all semesters of both SGPA and CGPA.

  • CGPA DATA


    The stored CGPA appears where the user can either delete it or download the same.

  • SGPA DATA


    The stored SGPA appears where the user can either delete it or download the same.
